# PyBank - Simple CLI Banking System

## Description
This is a simple command-line banking system built in Python.  
It allows users to:
- Create accounts
- Check account balances
- Deposit money
- Withdraw money
- View transaction history

All account and transaction data are stored locally in JSON files.

## Features
- CLI-based interface
- Persistent data using JSON
- Transaction logging (deposit & withdrawal)
- Unique 9-digit account numbers
